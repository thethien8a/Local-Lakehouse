Hệ thống là gì?

Một nền tảng Data Lakehouse end-to-end, xử lý dữ liệu chuyến taxi vàng (Yellow Taxi) của New York City — từ ingest dữ liệu thô đến dashboard phân tích.

Medallion Architecture
ACID Transactions
Automated ETL
Real-time Monitoring

🗄️ Apache Iceberg

Open table format hỗ trợ ACID transactions, time travel, schema evolution trên data lake

⚡ Apache Spark

Distributed processing engine xử lý ETL cho cả 3 tầng Bronze, Silver, Gold

🔄 Apache Airflow

Orchestration engine điều phối pipeline, SSH submit Spark jobs, trigger DAG chain

🏷️ Project Nessie

Git-like catalog cho Iceberg — hỗ trợ branching, merging, WAP pattern cho data quality

📦 MinIO

S3-compatible object storage lưu trữ data files và warehouse cho Iceberg tables

🔍 Trino

Distributed SQL query engine đọc/phân tích dữ liệu từ Iceberg tables qua Nessie catalog

📊 Metabase

BI Dashboard trực quan hoá dữ liệu taxi — KPI, heatmap, scatter plot, bar chart

📈 Prometheus + Grafana

Monitoring toàn stack — Spark, Airflow, MinIO, PostgreSQL metrics & alerting
