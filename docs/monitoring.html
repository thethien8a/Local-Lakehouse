📊 Grafana Dashboards

🏠 Lakehouse Overview

Tổng quan toàn hệ thống — sức khỏe services, data throughput, storage usage

⚡ Apache Spark Performance

Executor metrics, JVM memory, task duration, shuffle read/write

🔄 Airflow Orchestration

DAG run success/fail rate, task duration, scheduler heartbeat, pool usage

📦 MinIO Storage

Bucket size, object count, API latency, network traffic, capacity

⚠️ MinIO metrics yêu cầu bearer token — tự động generate bởi minio-init container.

🔔 Alert Rules

# conf/prometheus/alerts/lakehouse.yml # Các rule alert quan trọng: - Spark job failed / stuck - MinIO disk space low - Airflow scheduler heartbeat missing - PostgreSQL connections high
