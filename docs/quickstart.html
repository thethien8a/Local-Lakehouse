Getting Started

Quick Start

Khởi chạy toàn bộ hệ thống trong vài bước

⚙️ Yêu cầu hệ thống: Docker Engine 24+, Docker Compose v2, tối thiểu 8 GB RAM (khuyến nghị 16 GB), 20 GB ổ đĩa trống.

1. Clone repository

git clone <repo-url> && cd lakehouse

2. Cấu hình môi trường

cp .env.example .env — Điền credentials cho MinIO, PostgreSQL, Airflow, Grafana

3. Khởi chạy toàn bộ stack

docker compose up -d — 13 services sẽ tự động khởi động

4. Đợi services sẵn sàng

Kiểm tra: docker compose ps — Đợi tất cả status = Running

5. Chuẩn bị dữ liệu

Đặt file Parquet vào data/, chạy split_data_by_day.py để tạo Landing Zone

6. Chạy ETL Pipeline

Vào Airflow UI (port 8080), trigger DAG bronze_ingestion với params date_from/date_to

7. Xem dashboard

Metabase (port 3000) — kết nối Trino (port 8083) để truy vấn Iceberg tables
